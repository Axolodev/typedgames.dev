## Cambiando el contenido del documento

Para empezar a dibujar necesitamos definir un espacio que nos permita hacer esto, algo similar a un lienzo. Para esto, cambiaremos el `h1` que creamos en el post anterior por un elemento `canvas` que nos permitirá dibujar dentro de nuestro documento; el canvas debe tener una altura y un ancho definido, por lo que le asignaremos un espacio de 400 pixeles de ancho por 400 de alto. El `body` de nuestro HTML debería quedar algo así:

```html
<body>
  <canvas width="640" height="480"></canvas>
  <script src="./src/index.ts"></script>
</body>
```

Sin embargo, si ejecutamos esto en nuestro navegador nos encontraremos con una pantalla en blanco, lo que dificultará un poco nuestro trabajo. Podemos cambiar el estilo del canvas asignándole un color de fondo que nos permita verlo con más facilidad... pero no soy muy fan de esto porque puede provocar problemas de contraste en un futuro. A mi preferencia, es más fácil si asignamos una sombra al canvas y lo centramos en la página. 



<CodeWave>

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Desarrollo de videojuegos</title>
    <style>
      body {
        margin: 0;
      }

      .flex {
        display: flex;
        align-items: center;
        justify-content: center;
        min-height: 99vh;
      }

      .small-shadow {
        box-shadow: 0 2px 6px 0 hsla(0, 0%, 0%, 0.2);
      }
    </style>
  </head>

  <body>
    <main class="flex">
      <canvas
        id="game-area"
        width="400"
        height="400"
        class="small-shadow"
      ></canvas>
    </main>
    <script src="./src/index.ts"></script>
  </body>
</html>
```
